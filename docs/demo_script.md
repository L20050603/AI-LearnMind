# AI-LearnMind 期末大作业演示脚本

本脚本用于展示“AI-LearnMind 知学伴：基于专家系统、知识图谱与情感计算的个性化学习诊断与情智一体陪伴机器人原型”。

## 1. 启动项目

后端：

```bash
cd backend
venv\Scripts\activate
python seed.py
uvicorn main:app --reload --host 127.0.0.1 --port 8000
```

前端：

```bash
cd frontend
npm run dev
```

打开 `http://localhost:5173`，使用 `demo / 123456` 登录。

## 2. 最终演示路线

1. 登录 demo，进入 Dashboard，说明系统定位：学习诊断、专家规则、知识图谱、情绪陪伴和 Agent 黑板协同。
2. Dashboard 展示“机器智能创新设计演示”入口，说明完整闭环。
3. 设置页展示学习主题可切换、可新增、可编辑，说明 CoursePack 是可扩展知识库。
4. 学习地图展示“人工智能与机器智能基础”主题，说明关卡来自知识图谱与掌握度计算。
5. 知识图谱展示知识表示、前置关系、相关关系和易混淆关系。
6. 风险中心展示专家系统五部分：知识库、工作记忆、推理机、解释器、用户接口，并讲解推理链。
7. Agent 实验室展示多 Agent 黑板协同：画像、诊断、规划、情绪、干预、报告。
8. LearnMind Bot 输入“我快考试了很焦虑，专家系统总是记不住”，展示情智一体机器人感知—判断—动作—反馈机制。
9. 资源猎手搜索“专家系统”，展示本地资料 fallback、官方搜索 API 配置和合规 URL 抓取说明。
10. 测验页面展示答题后影响学习记录、掌握度、XP 和风险评分。
11. 报告页生成学习周报，并点击“生成创新设计素材”，复制 Markdown。
12. 创新设计中心展示设计初衷、与已有产品不同、核心功能、主要智能技术、系统框架、工作原理和创新点。

## 3. 讲解重点

- 专家系统：风险中心不是只给分数，而是展示知识库、工作记忆、推理机、解释器和用户接口。
- 知识图谱：学习地图、知识图谱和知识星图都由课程包知识点驱动。
- 情智一体：LearnMind Bot 同时处理认知状态和情绪状态，给出安抚与学习行动。
- 黑板机制：多个 Agent 写入共享黑板，最终由综合决策器形成建议。
- 无 API Key 可运行：系统本地规则、题库、资料检索和模板回复均可演示。
