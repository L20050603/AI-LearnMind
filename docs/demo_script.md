# AI-LearnMind 演示脚本

本脚本用于课堂展示“学习记录 → 风险判断 → Agent 协同 → 路径规划 → 报告生成”的完整闭环。

## 1. 准备演示数据

```bash
cd backend
venv\Scripts\activate
python seed.py
uvicorn main:app --reload --host 127.0.0.1 --port 8000
```

```bash
cd frontend
npm run dev
```

打开：

```text
http://localhost:5173
```

## 2. 展示学习闯关地图

1. 进入首页后，先说明顶部导航：学生、目标、等级、XP。
2. 指向中央“学习闯关地图”，说明 8 个操作系统知识点来自本地知识图谱。
3. 点击“页面置换算法 Boss”，展示：
   - Boss 状态
   - 前置知识
   - 掌握度
   - 推荐学习策略
4. 滚动到 React Flow 知识图谱，展示前置关系和关卡解锁逻辑。

## 3. 添加学习记录

1. 在“学习数据闭环”区域点击“添加学习记录”。
2. 选择知识点 ID `6`，填写：
   - 学习时长：`40`
   - 正确数：`3`
   - 错误数：`5`
   - 备注：`LRU 与 FIFO 仍容易混淆`
3. 提交后观察：
   - 今日/本周学习时长更新
   - 闯关地图掌握度变化
   - 风险中心评分变化

## 4. 触发风险预警

1. 点击“添加情绪打卡”。
2. 输入：
   - 心情：`焦虑`
   - 文本：`页面置换一直算错，很疲惫，需要帮助`
3. 提交后展示风险中心：
   - 风险分
   - 触发规则
   - 风险原因
   - 建议卡片

## 5. 运行多 Agent 协同

1. 在右侧 AI 导师面板点击“Agent 分析”。
2. 依次展示：
   - Profile Agent：学习画像
   - Diagnosis Agent：薄弱知识点与错题区域
   - Planner Agent：今日路线
   - Emotion Agent：压力与触发词
   - Intervention Agent：干预建议
   - Report Agent：阶段总结
3. 最后展示“最终综合建议”，说明它来自黑板内容和置信度加权。

## 6. 生成学习报告

1. 点击“生成周报”。
2. 展示周报卡片：
   - 本周学习总时长
   - 任务完成率
   - Agent 综合判断
3. 点击“复制 Markdown”，说明可以直接提交或放入 Word/PDF 报告。

## 7. AI 导师连续问答

可输入：

```text
LRU 和 FIFO 有什么区别？
```

再继续追问：

```text
能给我一个页面置换做题步骤吗？
```

说明：如果没有配置 LLM API Key，系统会使用本地课程资料关键词检索和模板生成回复；如果配置了 `OPENAI_API_KEY` 或 `LLM_API_KEY`，则尝试调用真实 LLM。
